折腾了半天终于从 Halo 换到了 Hugo。Halo 上没有一个能够直接导出为静态文章的插件，只能手动复制内容。还好我大部分的内容基本都是 Markdown 编辑器写的，不然一堆排版直接丢了 :(。

Halo 是真不适合我这种小站点使用，占用资源多，并且我的需求简单到只需要把内容和图片展示出来就行。因此在 2026 年，我对博客站的第一个改动就是迁移到静态博客。通过 GitHub 仓库托管博客文章，并使用 Cloudflare CDN 分发博客内容。这样既节约服务器资源，又省去了服务器维护的麻烦。~~我怎么不早点换用 Serverless 呢？~~
