我的博客网站
date_range发布于 2023-06-22 | list标签：小站日志
tang_ge

前言

此博客依靠 Hexo 进行搭建。本博客使用的主题是 mduilw。这个主题已经停止更新了，并且存在很多问题，因此本博客使用的使其魔改后的版本。

我原本是打算将博客部署在热铁盒上，但迫于其限制较多，最终选择了折中的方案–也就是部署在了 GitHub Page 上。只要 GitHub 不抽风，世界的大门仍旧敞开着。

更新计划

我会先完善博客的浏览体验。由于太久不写 HTML + JavaScript + CSS 了，网页中的很多东西会写 UI 方面的错误。不过在 ChatGPT 与 文心一言 的帮助下，网页修复效率提高了很多，估计过段时间就可以得到很不错的浏览体验了。

会写软件的跟新日志、软件一些功能的实现方式、自己玩的小算法等等。当然更新不会太频繁，我毕竟还是一名学生党，仍旧有很多的东西需要去学习，进步的空间还有许多。

后言

这篇博客的封面是 明日方舟 里的，这是一个很优秀的游戏，有空可以去试玩一下😊
